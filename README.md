# Commute-calc (通勤交通費 コピペくん)

毎月の交通費申請、地味に面倒くさいですよね。
「何日出社したっけ？」「合計金額いくら？」「Concurにコピペするtext作って」を解決するために自分用に作りました。

## 🔗 動いてるやつ (Demo)
ブラウザからここで使えます：
https://commute-calc-xzqi.onrender.com/

## 🛠 これで何ができる？
- **カレンダーをポチポチするだけ**: 月を選ぶとカレンダーが出るので、出社日をチェック。
- **自動計算**: 片道運賃を入れると、往復分を勝手に合算します。
- **そのままコピペ**: 精算ツールに貼り付けるための「◯月：1, 2, 8... 合計◯◯円」的なテキストを吐き出します。

## 🚀 使い方
1. 年月を選ぶ。
2. 出発駅・到着駅・片道運賃を入れる（Kuhombutsu <-> Roppongi はデフォルトなので書き換えてね）。
3. 出社日をチェック。
4. 下のテキストエリアに出る文字をコピーして、社内ツールにペースト！
